    # 影像产品DIY工具说明

    # 客户端（webapp）

    • 依赖：jquery-3.2.1
    • UI：Amaze UI
    • 图像处理：SVG.js 、svg.draggable.js
    • 颜色选择器：jscolor
    • 截图模块：Cropper.js
    • 字体大小选择Picker：picker
    • 截图模块：Cropper.js
    • 合图工具：Canvg.js

    # PC 设计后台：

    • 底层依赖：jquery-3.2.1
    • UI：layui
    • 图像处理：SVG.js 、svg.draggable.js、svg.resize.js、svg.select.js
    • 颜色选择器：Really Simple Color Picker in jQuery
    • 截图模块：Cropper.js
    • 合图工具：Canvg.js
